# Slither Grove 1.0

A Snake game built with C++ and SFML 3.1.0

## Features

- Grid-based movement
- Apple spawning
- Snake growth
- Wall collision
- Self collision
- Game Over state

## Controls

- Arrow Keys - Move the snake

## Technologies

- C++
- SFML 3.1.0
- Visual Studio 2026

## Planned Features

- Score display
- Pixel art graphics
- Sound effects
- Restart button
- Main menu
