# Slither Grove (Snake Game)

A Snake game built with C++ and SFML 3.1.0

## Features

- Grid-based movement
- Apple spawning
- Snake growth
- Wall collision
- Self collision
- Game Over state
- Score display
- Restart button (R button to restart if game over)
- Pixel art graphics
- Snake eating simple animation
- Sound Effects

## Controls

- Arrow Keys to move the snake

## Technologies

- C++
- SFML 3.1.0
- Visual Studio 2026

## Planned Features
